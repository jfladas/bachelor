# Amorphous Blob

_Bachelor 2026_

### [Download for Windows or MacOS](https://www.jfladas.com/projects/bachelor)

Desktop presence built with Vue, Vite, Electron, and Matter.js.

## The Experience

A continuous, subtle relationship with a digital representation of yourself that takes form through your input, then remains present peripherally, reflecting your personality and mood through color, motion, and interaction.

## Development

- Start app (renderer + Electron): `npm run dev`
- Build packaged app (Windows): `npm run build`
- For MacOS build see [docs/MAC_BUILD.md](docs/MAC_BUILD.md)
