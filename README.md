# My Self Companion

Desktop companion built with Vue, Vite, Electron, and Matter.js.

## The Experience

My Self Companion creates a continuous, subtle relationship with a digital representation of yourself. It takes form through your input, then remains present as a companion that reflects aspects of that profile through color, motion, and interaction.

## What It Entails

- A guided introduction that gathers traits, preferences, and a color identity.
- A companion presence that responds to touch, movement, and the edges of the desktop.
- A shifting visual mood shaped by the profile it learns from you.
- A reflective journal space for short notes, prompts, and emotional context.
- A small set of companion interactions that keep the experience close at hand without getting in the way.
- Saved state that lets the relationship continue across sessions.

## Development

- Start app (renderer + Electron): `npm run dev`
- Build packaged app: `npm run build`
